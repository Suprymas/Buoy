In this folder not much was done. Initial coding was started to track the cloud movements but no more.

If you are starting here your next steps should be to gather some training data with the Esp32 camera and train a model.

Then use "anypoint tracking" method or others to detect the movements of the sky.

In the [camera_calibrate.py](./camera_calibrate.py) file is the code for calibrating the camera of Esp32.

In the [cloudDetectionOpticalFlow.py](./cloudDetectionOpticalFlow.py) was my first try at trying to detect cloud movements.
