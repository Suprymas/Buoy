# 3D models

## Formats

The files are provided in three formats: SLDPRT, STEP and STL.

SLDPRT - Contains SOLIDWORKS (2025 SP5 version) files. Used to modify the models in SOLIDWORKS.

STEP - Contains .STEP files. Used to slightly modify files with other programs.

STL - Containts printing files. Used for printing the models.

## Files

body - the main buoy body on which the XIAO ESP32S3 case, GPS, and wind sensor are mounted.

body-bottom - glued to the bottom of the body, houses the 18650 cell casing, and has the bearing inserted into the bottom.

esp-case - houses the XIAO ESP32S3. Has holes for pins, antenna, and USB-C.

esp-case-top - closes the XIAO ESP32S3 into the case, has a hole for the camera, and a special place to glue the lens on.

handle - used to attach the whole thing to the pole.
